Incident Handling Process:

Introduction
1. Incident Handling:
- Ein Event ist eine Aktion die in einen System stattfindet
- Ein Incident ist ein Ereignis mit negativen Konsequezen wie Datendiebstahl, Unbefugter Zugriff auf Daten usw.

2. Cyber Kill Chain:
- Die Cyber Kill Chain gliedert sich in 7 Schritte auf
1. Recon
	- Informationssammlung von Daten von dem Angreifer
2. Weaponize
	- Der Angreifer erstellt eine Exploit fuer das System
3. Deliver
	- Der Angreifer sendet den Exploit an das System
4. Exploit
	- Der Exploit wird ausgeloest von dem Opfer
5. Install
	- Der Exploit wird auf dem System installiert
6. C&C
	- Der Angreifer erhaelt eine Remote Access auf das kompremierte System
7. Action
	- Hier kann der Angreifer auf dem System Aktionen ausfuehren
